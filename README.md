jquery.qrcode
=============

jQuery QR-Code-Plugin

Example usage
=========================================

```js

	$(document).ready(function() {
		$('#test').qrcode({
			'url' : 'http://jquery.com',
			'width' : 300,
			'height' : 300,
			'qrsize' : 100
		});
	});

```

This code will add a QR-Code to the div '#test'.
